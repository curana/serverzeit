Host Europe: Rootserver M

Bestellprozess

Der Bestellprozess von Host Europe ist erwähnenswert, da umfangreicher und einer der wenigen mit "Papierkram". Host Europe stellt per Post einen Aktivierungscode zu, mit dem der Zugang zum Kunden Informationssystem (KIS) zunächst freigeschaltet werden muss.

Nach der Freischaltung dauert es dann erneut ein paar Tage (gemäss Support 4-5 Werktage), bis der Server bereitgestellt ist. Ein Upgrade der Ressourcen will daher geplant sein, wenn auch die erneute Aktivierung auf dem Postwege bei einem zusätzlichen Server entfallen dürfte.

KIS und Installation

Das Kunden-Informationssystem (KIS) ist unglaublich unübersichtlich. Ich ging davon aus, dass ich mich in den meisten Anwendungen schnell zurecht finde, ich habe mich allerdings getäuscht. Es ist nicht hässlich, aber es wirkt so konfus. Punkte sind nicht da wo man sie erwartet. Hier würde ich mir eine aufgeräumtere Struktur und klarere Benennung der einzelnen Menüpunkte wünschen.

Unter "Produkte" bspw. würde ich entweder die HE-Produkte oder aber meine bei HE gekauften/gemieteten Produkte erwarten. Hier finden sich allerdings Softwareprodukte u.a., die über HE günstiger bezogen werden können. Diesen Punkt könnte man daher in "Sonderangebote" oder "Rabattaktionen" oder sowas umbenennen.

Host Europe wirbt damit, dass auch eigene ISO-Images für die Installation eigener Betriebssysteme hochgeladen und gemountet werden können. Bei mir hat dieser Prozess leider nicht funktioniert, der Support hat das Image dann von der FreeBSD-Seite geladen und das ISO gemountet. Anschliessend konnte ich via VNC auf den Server zugreifen und so die Installation problemlos durchführen, als würde ich vor dem Rechner sitzen. Ich persönlich bin grosser Fan von Remote-Installationen und würde sie immer einer Image-Installation vorziehen.

Performance

Die Root-Server von Host Europe sind "dedizierte Bare Metal Virtual Server". Das heisst, dass ein dedizierter Server aufgestellt wird, auf dem dann eine Virtualisierungssoftware installiert wird (Parallels Bare Metal Server) und darauf dann ein einziger vServer betrieben wird. Somit hat man fast (!) die Leistung eines dediziertern Servers, allerdings mit den Vorteilen einer virtuellen Umgebung (Portierbarkeit etc.).

Der Ubench-Test ergab folgende Werte, die etwas unter denen des vergleichbaren Webtropia Professional M liegen:

Ubench CPU: 1036389
Ubench MEM:  326449
--------------------
Ubench AVG:  681419

Etwas irritierend ist die Angabe bzgl. der Bandbreitennutzung. Host Europe gibt je nach Paket an, dass eine gewisse Peak-Bandbreite garantiert ist. Wird diese allerdings regelmässig überschritten, wird diese begrenzt und kann nach Rücksprache wieder freigegeben werden. Ich konnte in meinem Test nicht feststellen was das bedeutet, bei grösseren Projekten würde ich das aber abklären und am besten schriftlich festhalten wie das genau geregelt wird.

Service und Support

Der Service und Support von Host Europe ist tadellos. Sicherlich wären kürzere Bereitstellungszeiten wünschenswert (siehe Hetzner), aber in der Regel haben professionell aufgezogene Projekte keine geplante Vorlaufzeit von 2 Tagen. Dennoch wäre eine grössere Transparenz diesbezüglich wünschenswert.

Die Antworten waren präzise auf meine Fragen ausgerichtet, waren hilfreich und durchaus kompetent. Während meines Kurztests hat Host Europe jedenfalls seinen guten Ruf in Sachen Service bestätigt.

Fazit

Insgesamt bin ich von den Servern begeistert. Zu einem fairen Preis bietet Host Europe einen leistungsfähigen Server an, der alles mitbringt was benötigt wird. Interessant sind vor allem die Backups, die sich vollautomatisiert im Hintergrund anlegen lassen. Das gibt ein dickes Plus.

Was mich etwas gestört hat ist, dass auf der Seite mit "Upgradebarkeit" geworben wird, da der Server ja virtualisiert ist. Auf Nachfrage beim Support hin wurde mir allerdings mitgeteilt, dass das aktuell nicht möglich ist. Zwar wird ein grösserer Server parallel bereitgestellt, sodass ein Umzug möglich ist, allerdings musste ich mir dann folgendene Frage stellen:

"Welchen Vorteil hat ein virtueller Server gegenüber einem dedizierten Server, wenn der Vorteil der Upgrade- und Übtragbarkeit nicht gegeben ist?"
