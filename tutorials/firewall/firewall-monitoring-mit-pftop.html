Firewall-Monitoring mit pftop

Du kennst vielleicht schon das Programm "top" mit dem du die aktuelle Serverauslastung einsehen kannst. Für pf gibt es ein ähnliches Tool, welches "pftop" heißt. Mit diesem Tool kannst du dir den aktuellen Zustand deiner Firewall, und noch einiges mehr, einsehen.

  1. Zunächst müssen wir das Tool installieren:

    # cd /usr/ports/sysutils/pftop/ && make install clean

  2. Wenn du nun in der Konsole ein

    # pftop

    eingibst, wird dir eine Liste aktueller Verbindungen angezeigt. Hier kannst du sehen welche IP die Adresse hergestellt hat und auf welchen Port sie zugreift. Die Ausgabe ist eigentlich recht selbsterklärend.

  3. Mit der Taste 'q' beendest du pftop wieder.
